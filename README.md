## Понятие алгоритма
Алгоритм - это предписания, задающие вычислительный процесс, идущий от варьируемых начальных данных к искомому результату. Свойства алгоритма:
1. Однозначность
2. Дискретность
3. Универсальность
4. Детерминированность
5. Результативность
